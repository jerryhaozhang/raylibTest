# Raylib C Project

This is a simple C project using the raylib library for game development.

## Prerequisites

- Install raylib: Download from https://www.raylib.com/ and follow installation instructions, or use Homebrew: `brew install raylib` (on macOS). If Homebrew is not installed, install it first.

## Build

Run `make` to build the project.

## Run

Run `./game` to start the application.

## Clean

Run `make clean` to remove build artifacts.
